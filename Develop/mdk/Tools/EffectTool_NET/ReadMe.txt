========================================================================
    응용 프로그램 : EffectTool_NET 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 EffectTool_NET 응용 프로그램을 만들었습니다.  

이 파일에는 EffectTool_NET 응용 프로그램을 구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.

EffectTool_NET.vcproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 
    파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한
    플랫폼, 구성 및 프로젝트 기능에 대한 정보가 포함되어 있습니다.

EffectTool_NET.cpp
    기본 응용 프로그램 소스 파일입니다.
    폼을 표시하기 위한 코드가 포함되어 있습니다.

Form1.h
    폼 클래스와 InitializeComponent() 함수의 구현이 포함되어 있습니다.

AssemblyInfo.cpp
    어셈블리 메타데이터를 수정하는 데 필요한 사용자 지정 특성이 포함되어 있습니다.

/////////////////////////////////////////////////////////////////////////////
기타 표준 파일:

StdAfx.h, StdAfx.cpp
    이 파일은 미리 컴파일된 헤더(PCH) 파일 EffectTool_NET.pch와
    미리 컴파일된 형식(PCT) 파일 StdAfx.obj를 빌드하는 데 사용됩니다.
